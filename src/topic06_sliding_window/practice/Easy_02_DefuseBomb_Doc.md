# Easy_02_DefuseBomb — Довідка

## Техніка: Circular Window via Modulo Indexing

**Суть:** вікно фіксованого розміру `k`, що "обгортається" навколо
масиву. Замість перевірки меж вручну, індекс елемента, що на `d`
кроків попереду/позаду позиції `i`, обчислюється модульною
арифметикою: `(i + d) % n` (з урахуванням `n`, щоб уникнути від'ємного
залишку в Java: `((i + d) % n + n) % n`).

```java
int n = code.length;
int[] result = new int[n];
for (int i = 0; i < n; i++) {
    int sum = 0;
    for (int d = 1; d <= k; d++) {
        sum += code[((i + d) % n + n) % n]; // circular forward lookup
    }
    result[i] = sum;
}
```

**Коли застосовувати:** масив трактується як "кільце" — потрібно
підсумувати/агрегувати сусідні елементи, які можуть виходити за межі
масиву з одного кінця і продовжуватись з іншого (розклад, дні тижня,
кругові черги).

**Складність:** O(N*K) наївно (можна оптимізувати до O(N) через ту саму
ідею ковзного вікна з O(1) оновленням, застосовану по колу).
