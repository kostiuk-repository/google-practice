# Easy_01_MaxAverageSubarray — Довідка

## Техніка: Fixed-Size Sliding Window (O(1) Sum Update)

**Суть:** тримаємо вікно фіксованого розміру `k`. Замість перерахунку
суми вікна щоразу (`O(k)`), оновлюємо її за `O(1)`: при зсуві вікна на
одну позицію вправо додаємо елемент, що входить, і віднімаємо елемент,
що виходить.

```java
int windowSum = 0;
for (int i = 0; i < k; i++) windowSum += nums[i];
double best = (double) windowSum / k;
for (int i = k; i < nums.length; i++) {
    windowSum += nums[i] - nums[i - k];       // O(1) update
    best = Math.max(best, (double) windowSum / k);
}
```

**Коли застосовувати:** розмір вікна ФІКСОВАНИЙ і заданий заздалегідь
(`k`), а треба знайти найкраще (макс/мін) значення суми, середнього чи
іншої адитивної метрики серед усіх вікон цього розміру.

**Складність:** O(N) час (один прохід + O(1) на крок), O(1) пам'ять
(окрім вихідних даних).
