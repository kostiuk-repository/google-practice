# Easy_05_MaxVowels — Довідка

## Техніка: Fixed-Size Window + Running Counter

**Суть:** вікно фіксованого розміру `k` ковзає рядком; замість
перерахунку кількості голосних у вікні щоразу, тримаємо лічильник, який
оновлюємо за O(1): +1, якщо символ, що входить у вікно, голосний,
-1, якщо символ, що виходить з вікна, був голосним.

```java
int count = 0;
for (int i = 0; i < k; i++) if (isVowel(s.charAt(i))) count++;
int best = count;
for (int i = k; i < s.length(); i++) {
    if (isVowel(s.charAt(i))) count++;
    if (isVowel(s.charAt(i - k))) count--;
    best = Math.max(best, count);
}
```

**Коли застосовувати:** потрібно порахувати/максимізувати кількість
символів певного типу (голосні, цифри, певна буква) у вікні
фіксованого розміру, що ковзає рядком.

**Складність:** O(N) час, O(1) пам'ять.

## Drills

1. **Easy_05_01** — змінений предикат: чи ВСІ символи вікна голосні (а не скільки).
2. **Easy_05_02** — дзеркальний лічильник: приголосні замість голосних.
3. **Easy_05_03** — узагальнення розміру вікна: діапазон [kMin, kMax] замість одного k.
4. **Easy_05_04** — трекінг індексу найкращого вікна (argmax), а не лише кількості.
5. **Easy_05_05** — капстоун: дзеркальний напрямок (мінімум) + повернення тексту підрядка.
