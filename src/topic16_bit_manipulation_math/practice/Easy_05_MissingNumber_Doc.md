# Easy_05_MissingNumber — Довідка

## Техніка: XOR сума / Сума Гаусса

**Суть:** Знаходження пропущеного елемента в послідовності порівнянням очікуваної суми чисел `[0..N]` з фактичною сумою масиву, або через побітовий XOR.

```java
int missing = nums.length;
for (int i = 0; i < nums.length; i++) {
    missing ^= i ^ nums[i];
}
return missing;
```

**Коли застосовувати:** Дано масив довжини N з числами в діапазоні `[0..N]`, одне число пропущено.

**Складність:** O(N) час, O(1) додаткова пам'ять.

## Drills

1. **Easy_05_01** — пошук пропущеного числа в діапазоні від 1 до N+1.
2. **Easy_05_02** — пошук пропущеного числа в діапазоні від 0 до N за допомогою XOR.
3. **Easy_05_03** — пошук дубліката та пропущеного числа одночасно (set error numbers).
4. **Easy_05_04** — пошук пропущеного числа у відсортованому масиві за O(log N).
5. **Easy_05_05** — знаходження двох пропущених чисел у діапазоні від 1 до N.

Розв'язуй у порядку 01 → 05, не підглядаючи в Easy_05_MissingNumber.java.
