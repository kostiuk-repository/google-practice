# Easy_06_PowerOfTwo — Довідка

## Техніка: Побітова перевірка степеня двійки

**Суть:** Перевірка, чи має число рівно один встановлений біт за допомогою властивості `n & (n - 1) == 0` для `n > 0`.

```java
return n > 0 && (n & (n - 1)) == 0;
```

**Коли застосовувати:** Перевірка степенів двійки, оптимізація розмірів буферів, швидке ділення/множення.

**Складність:** O(1) час, O(1) додаткова пам'ять.

## Drills

1. **Easy_06_01** — перевірка, чи є число степенем четвірки.
2. **Easy_06_02** — циклічна перевірка степеня двійки через ділення.
3. **Easy_06_03** — побітова перевірка степеня двійки з урахуванням граничних значень.
4. **Easy_06_04** — пошук найбільшого степеня двійки, який менший або дорівнює N.
5. **Easy_06_05** — пошук найменшого степеня двійки, який більший або дорівнює N.

Розв'язуй у порядку 01 → 05, не підглядаючи в Easy_06_PowerOfTwo.java.
