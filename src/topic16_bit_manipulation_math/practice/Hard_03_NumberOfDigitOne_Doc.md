# Hard_03_NumberOfDigitOne — Довідка

## Техніка: Порозрядний підрахунок цифр / Математичний аналіз розрядів

**Суть:** Для кожного розряду (одиниці, десятки, сотні тощо) обчислюється кількість разів, коли на цьому розряді з'являється цифра 1 у числах від 1 до N, на основі трьох частин: вищої, поточної та нижчої частин числа.

```java
int count = 0;
for (long i = 1; i <= n; i *= 10) {
    long divider = i * 10;
    count += (n / divider) * i + Math.min(Math.max(n % divider - i + 1, 0L), i);
}
return count;
```

**Коли застосовувати:** Підрахунок певної цифри в діапазоні від 0 до N для дуже великих N.

**Складність:** O(log_10(N)) час, O(1) додаткова пам'ять.
