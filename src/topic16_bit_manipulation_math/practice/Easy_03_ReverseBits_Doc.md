# Easy_03_ReverseBits — Довідка

## Техніка: Реверс бітів / Розділяй і володарюй

**Суть:** Перестановка бітів цілого числа у зворотному порядку за допомогою побітових зсувів та масок.

```java
int result = 0;
for (int i = 0; i < 32; i++) {
    result = (result << 1) | (n & 1);
    n >>>= 1;
}
return result;
```

**Коли застосовувати:** Низькорівневе кодування даних, обробка сигналів (FFT), криптографія.

**Складність:** O(1) час (фіксовані 32 кроки) або O(log W) де W — розмір машинного слова (через побітове маскування).

## Drills

1. **Easy_03_01** — реверс лише молодших 8 бітів 32-бітного числа.
2. **Easy_03_02** — обмін місцями старших 16 бітів та молодших 16 бітів.
3. **Easy_03_03** — реверс бітів для 16-бітного знакового числа (short).
4. **Easy_03_04** — перевірка, чи є 32-бітне двійкове представлення числа паліндромом.
5. **Easy_03_05** — реверс усіх 32 бітів числа без використання циклу (через маски).

Розв'язуй у порядку 01 → 05, не підглядаючи в Easy_03_ReverseBits.java.
