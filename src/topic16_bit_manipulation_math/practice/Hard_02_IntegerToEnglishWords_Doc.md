# Hard_02_IntegerToEnglishWords — Довідка

## Техніка: Розбиття на класи чисел та словникове відображення

**Суть:** Розбиття числа на трійки цифр (мільйарди, мільйони, тисячі, одиниці) та рекурсивна обробка кожної трійки за допомогою константних словникових масивів.

```java
// Псевдокод відображення
private final String[] LESS_THAN_20 = {"", "One", "Two", ...};
private final String[] TENS = {"", "Ten", "Twenty", ...};
private final String[] THOUSANDS = {"", "Thousand", "Million", "Billion"};

public String numberToWords(int num) {
    if (num == 0) return "Zero";
    int i = 0;
    String words = "";
    while (num > 0) {
        if (num % 1000 != 0) {
            words = helper(num % 1000) + THOUSANDS[i] + " " + words;
        }
        num /= 1000;
        i++;
    }
    return words.trim();
}
```

**Коли застосовувати:** Перетворення чисел у словесну форму, генерація фінансових документів.

**Складність:** O(log_1000(N)) час (фактично O(1), бо обмежено 2^31-1), O(1) додаткова пам'ять.
