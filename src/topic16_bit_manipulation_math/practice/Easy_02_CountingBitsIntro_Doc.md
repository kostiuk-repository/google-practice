# Easy_02_CountingBitsIntro — Довідка

## Техніка: Побітове динамічне програмування

**Суть:** Використання зв'язку між числом та його зсунутою версією для побудови масиву кількості бітів за лінійний час: `dp[i] = dp[i >> 1] + (i & 1)`.

```java
int[] ans = new int[n + 1];
for (int i = 1; i <= n; i++) {
    ans[i] = ans[i >> 1] + (i & 1);
}
return ans;
```

**Коли застосовувати:** Необхідно згенерувати кількість встановлених бітів для послідовності чисел від 0 до N за O(N).

**Складність:** O(N) час, O(1) додаткова пам'ять (поза вихідним масивом).
