# Easy_10_Base7 — Довідка

## Техніка: Перетворення систем числення (Ділення та остача)

**Суть:** Перетворення десяткового числа в систему числення з основою B шляхом послідовного ділення на B та накопичення остач у зворотному порядку.

```java
if (num == 0) return "0";
StringBuilder sb = new StringBuilder();
boolean negative = num < 0;
num = Math.abs(num);
while (num > 0) {
    sb.append(num % 7);
    num /= 7;
}
if (negative) sb.append("-");
return sb.reverse().toString();
```

**Коли застосовувати:** Перетворення чисел у будь-яку іншу систему числення (двійкову, вісімкову, шістнадцяткову, 7-кову тощо) з підтримкою знаків.

**Складність:** O(log_B(N)) час, O(log_B(N)) додаткова пам'ять для результуючого рядка.

## Drills

1. **Easy_10_01** — перетворення додатного числа в двійковий рядок без вбудованих функцій.
2. **Easy_10_02** — декодування 7-кового рядка назад у десяткове ціле число.
3. **Easy_10_03** — перетворення десяткового числа в 16-ковий рядок (lowercase).
4. **Easy_10_04** — перетворення десяткового числа в рядок з довільною основою B (2 <= B <= 10).
5. **Easy_10_05** — конвертація рядка з однієї системи числення B1 в іншу B2.

Розв'язуй у порядку 01 → 05, не підглядаючи в Easy_10_Base7.java.
