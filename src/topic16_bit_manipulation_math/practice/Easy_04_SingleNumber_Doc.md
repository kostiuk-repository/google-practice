# Easy_04_SingleNumber — Довідка

## Техніка: Властивості виключного АБО (XOR)

**Суть:** Застосування властивостей XOR: `x ^ x = 0` та `x ^ 0 = x`, що дозволяє знайти унікальний елемент серед дублікатів.

```java
int result = 0;
for (int num : nums) {
    result ^= num;
}
return result;
```

**Коли застосовувати:** Масив містить пари однакових елементів і один унікальний. Потрібно знайти унікальний за O(N) часу та O(1) додаткової пам'яті.

**Складність:** O(N) час, O(1) додаткова пам'ять.

## Drills

1. **Easy_04_01** — обчислення XOR-суми всіх елементів масиву.
2. **Easy_04_02** — пошук унікального символу в масиві символів char.
3. **Easy_04_03** — пошук індексу унікального числа в масиві дублікатів.
4. **Easy_04_04** — пошук елемента, що входить непарну кількість разів у масив.
5. **Easy_04_05** — пошук двох унікальних чисел у масиві, де решта повторюються двічі.

Розв'язуй у порядку 01 → 05, не підглядаючи в Easy_04_SingleNumber.java.
