# Medium_01_SingleNumberII — Довідка

## Техніка: Порозрядне додавання за модулем K

**Суть:** Рахуємо суму бітів на кожній позиції по всьому масиву. Біти, сума яких не ділиться на 3, належать єдиному числу. Також можна спроектувати скінченний автомат з двох масок (`ones`, `twos`).

```java
int ones = 0, twos = 0;
for (int num : nums) {
    ones = (ones ^ num) & ~twos;
    twos = (twos ^ num) & ~ones;
}
return ones;
```

**Коли застосовувати:** Всі числа з'являються K разів (де K > 2, наприклад, 3), окрім одного, яке з'являється рівно 1 раз.

**Складність:** O(N) час, O(1) додаткова пам'ять.

## Drills

1. **Medium_01_01** — перевірка, чи встановлений k-й біт цілого числа.
2. **Medium_01_02** — підрахунок кількості чисел у масиві, які мають встановлений k-й біт.
3. **Medium_01_03** — пошук унікального додатного числа серед трійок дублікатів.
4. **Medium_01_04** — пошук унікального числа, якщо решта повторюється рівно чотири рази.
5. **Medium_01_05** — реалізація пошуку унікального числа серед трійок через оптимальні маски ones/twos.

Розв'язуй у порядку 01 → 05, не підглядаючи в Medium_01_SingleNumberII.java.
