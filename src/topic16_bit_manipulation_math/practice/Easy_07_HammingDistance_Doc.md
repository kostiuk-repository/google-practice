# Easy_07_HammingDistance — Довідка

## Техніка: Відстань Хеммінга / XOR та підрахунок одиниць

**Суть:** Відстань Хеммінга між двома числами дорівнює кількості бітів, у яких вони різняться, що обчислюється як кількість встановлених бітів у `x ^ y`.

```java
int xor = x ^ y;
int distance = 0;
while (xor != 0) {
    distance++;
    xor &= (xor - 1);
}
return distance;
```

**Коли застосовувати:** Алгоритми виявлення помилок (CRC), порівняння двійкових кодів, метрики подібності.

**Складність:** O(1) час (фіксована розрядність 32 біти), O(1) додаткова пам'ять.
