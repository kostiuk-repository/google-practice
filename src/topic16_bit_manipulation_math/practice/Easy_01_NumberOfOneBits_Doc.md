# Easy_01_NumberOfOneBits — Довідка

## Техніка: Побітові операції (Підрахунок встановлених бітів)

**Суть:** Побітовий зсув та маскування або використання операції `n & (n - 1)` для скидання наймолодшого встановленого біта (алгоритм Браяна Кернігана).

```java
int count = 0;
while (n != 0) {
    n = n & (n - 1); // скидаємо наймолодший встановлений біт
    count++;
}
return count;
```

**Коли застосовувати:** Потрібно порахувати кількість одиниць у двійковому представленні числа (Hamming weight), визначити парність бітів.

**Складність:** O(k) час, де k — кількість встановлених бітів (у найгіршому випадку O(W), де W = 32 біти), O(1) додаткової пам'яті.
