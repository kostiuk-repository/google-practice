# Medium_03_PowXN — Довідка

## Техніка: Швидке піднесення до степеня (Бінарне піднесення)

**Суть:** Обчислення x^N за O(log N) шляхом ділення показника навпіл: `x^n = (x^(n/2))^2` для парних, `x * x^(n-1)` для непарних, або ітеративно за допомогою бітів показника.

```java
double ans = 1.0;
long absN = Math.abs((long) n);
while (absN > 0) {
    if ((absN & 1) == 1) {
        ans *= x;
    }
    x *= x;
    absN >>= 1;
}
return n < 0 ? 1.0 / ans : ans;
```

**Коли застосовувати:** Швидке обчислення степенів великої розрядності, модульне піднесення, піднесення матриць.

**Складність:** O(log N) час, O(1) додаткова пам'ять для ітеративної версії.
