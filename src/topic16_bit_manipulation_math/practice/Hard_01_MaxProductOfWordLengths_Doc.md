# Hard_01_MaxProductOfWordLengths — Довідка

## Техніка: Бітові маски для символів / Перетин множин

**Суть:** Кодування символьного складу кожного слова в 32-бітове число. Символ `c` встановлює `c - 'a'`-й біт. Тоді порівняння `(maskA & maskB) == 0` перевіряє відсутність спільних літер за O(1).

```java
int n = words.length;
int[] masks = new int[n];
for (int i = 0; i < n; i++) {
    for (char c : words[i].toCharArray()) {
        masks[i] |= (1 << (c - 'a'));
    }
}
int maxProd = 0;
for (int i = 0; i < n; i++) {
    for (int j = i + 1; j < n; j++) {
        if ((masks[i] & masks[j]) == 0) {
            maxProd = Math.max(maxProd, words[i].length() * words[j].length());
        }
    }
}
return maxProd;
```

**Коли застосовувати:** Перевірка перетину символьних складів багатьох рядків, швидке порівняння підмножин.

**Складність:** O(L + N^2) час (де L — сумарна довжина всіх слів, N — кількість слів), O(N) додаткова пам'ять для збереження масок.
