# Easy_08_AddBinary — Довідка

## Техніка: Додавання у двійковій системі / Порозрядне додавання з переносом

**Суть:** Імітація додавання в стовпчик справа наліво: на кожній позиції додаємо символи рядків `a` та `b` разом із переносом `carry`.

```java
StringBuilder sb = new StringBuilder();
int i = a.length() - 1, j = b.length() - 1, carry = 0;
while (i >= 0 || j >= 0 || carry > 0) {
    int sum = carry;
    if (i >= 0) sum += a.charAt(i--) - '0';
    if (j >= 0) sum += b.charAt(j--) - '0';
    sb.append(sum % 2);
    carry = sum / 2;
}
return sb.reverse().toString();
```

**Коли застосовувати:** Робота з довгими числами, представленими як рядки з символів '0' та '1'.

**Складність:** O(max(N, M)) час, де N і M — довжини рядків; O(max(N, M)) додаткова пам'ять для результуючого рядка.

## Drills

1. **Easy_08_01** — додавання двійкових рядків однакової довжини без кінцевого переносу.
2. **Easy_08_02** — додавання поодиноких бінарних символів з урахуванням вхідного переносу.
3. **Easy_08_03** — додавання одиниці до двійкового рядка (інкремент).
4. **Easy_08_04** — віднімання одного двійкового рядка від іншого (a >= b).
5. **Easy_08_05** — додавання трьох двійкових рядків одночасно.
