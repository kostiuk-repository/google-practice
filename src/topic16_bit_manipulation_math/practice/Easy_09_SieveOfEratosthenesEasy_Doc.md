# Easy_09_SieveOfEratosthenesEasy — Довідка

## Техніка: Решето Ератосфена (Проста версія)

**Суть:** Алгоритм знаходження простих чисел до заданого N шляхом ітеративного викреслювання кратних чисел.

```java
boolean[] isPrime = new boolean[n];
Arrays.fill(isPrime, true);
int count = 0;
for (int i = 2; i < n; i++) {
    if (isPrime[i]) {
        count++;
        for (long j = (long) i * i; j < n; j += i) {
            isPrime[(int) j] = false;
        }
    }
}
return count;
```

**Коли застосовувати:** Потрібно швидко знайти або підрахувати прості числа в діапазоні від 2 до N.

**Складність:** O(N log log N) час, O(N) додаткова пам'ять для збереження стану викресленості.

## Drills

1. **Easy_09_01** — перевірка простоти одного числа методом перебору дільників.
2. **Easy_09_02** — генерація булевого масиву простоти чисел до N за допомогою решета.
3. **Easy_09_03** — знаходження всіх простих дільників числа.
4. **Easy_09_04** — пошук найменшого простого числа, яке строго більше N.
5. **Easy_09_05** — обчислення суми всіх простих чисел менших за N.
