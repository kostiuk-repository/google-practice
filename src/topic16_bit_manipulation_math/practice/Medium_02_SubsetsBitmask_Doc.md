# Medium_02_SubsetsBitmask — Довідка

## Техніка: Генерація підмножин за допомогою бітових масок

**Суть:** Для масиву довжини N кожна з 2^N підмножин відповідає двійковому числу від 0 до 2^N - 1. Якщо i-й біт числа встановлений, то i-й елемент входить у підмножину.

```java
List<List<Integer>> result = new ArrayList<>();
int n = nums.length;
for (int i = 0; i < (1 << n); i++) {
    List<Integer> subset = new ArrayList<>();
    for (int j = 0; j < n; j++) {
        if ((i & (1 << j)) != 0) {
            subset.add(nums[j]);
        }
    }
    result.add(subset);
}
return result;
```

**Коли застосовувати:** Генерація всіх підмножин (powerset) для невеликих N (N <= 15-20), повний перебір комбінацій.

**Складність:** O(N * 2^N) час, O(N * 2^N) додаткова пам'ять для результуючого списку підмножин.
