# Medium_04_SumOfTwoIntegers — Довідка

## Техніка: Побітовий суматор

**Суть:** Додавання двох чисел без знаків `+` та `-`: побітове XOR `a ^ b` обчислює суму без переносів, а AND зсув `(a & b) << 1` обчислює переноси. Вони додаються ітеративно, поки перенос не стане 0.

```java
while (b != 0) {
    int carry = (a & b) << 1;
    a = a ^ b;
    b = carry;
}
return a;
```

**Коли застосовувати:** Реалізація арифметики на низькому рівні, робота з обмеженнями компілятора/платформи.

**Складність:** O(1) час (максимум 32 ітерації для 32-бітного цілого), O(1) додаткова пам'ять.

3. **Medium_04_03** — перевірка, чи має сума чисел a та b непарну кількість встановлених бітів.
4. **Medium_04_04** — обчислення середнього значення двох чисел без переповнення.
5. **Medium_04_05** — множення двох чисел за допомогою зсувів та додавань (алгоритм селянина).
