# Dynamic Programming - Study Notes

**Динамічне програмування (DP)** використовується для вирішення задач шляхом розбиття їх на підзадачі, результати яких зберігаються (memoization/tabulation), щоб уникнути повторних обчислень.

---

## 1. Два підходи
* **Top-Down (Memoization):** Рекурсія з кешуванням.
* **Bottom-Up (Tabulation):** Ітеративна побудова таблиці від найпростіших станів до цільового.

---

## 2. Ключові етапи проектування DP
1. Визначити стан (state), що описує підзадачу.
2. Сформулювати перехід (transition) між станами.
3. Визначити базові випадки (base cases).
