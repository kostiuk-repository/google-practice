# Medium_04_PartitionEqualSubsetSum — Довідка

## Техніка: DP рюкзака (0/1 Knapsack Subset Sum DP)

**Суть:** Зведення задачі про поділ масиву до класичної задачі Subset Sum. Ми шукаємо підмножину елементів, сума яких дорівнює точно половині від загальної суми масиву. dp[j] представляє можливість отримати суму j за допомогою підмножини.

### Концепт
```java
// dp[j] = dp[j] || dp[j - num]
boolean[] dp = new boolean[target + 1];
dp[0] = true;
for (int num : nums) {
    for (int j = target; j >= num; j--) {
        dp[j] = dp[j] || dp[j - num];
    }
}
```

**Коли застосовувати:** При розділенні ресурсів на дві рівні (чи збалансовані) групи або при вирішенні задач вибору суми елементів без повторень.

**Складність:** Часова: O(N * S), де N — кількість чисел, S — половина суми (target). Просторова: O(S) додаткової пам'яті при оптимізованій 1D таблиці.

## Вправи (Drills)
- **01 (canPartitionBase)**: Перевірте можливість розділити масив на дві підмножини з рівною сумою, використовуючи двовимірний масив DP.
- **02 (canPartitionSpace)**: Реалізуйте Partition Equal Subset Sum з одновимірним масивом DP для економії пам'яті.
- **03 (minDifferencePartition)**: Поділіть масив на дві підмножини так, щоб абсолютна різниця їхніх сум була мінімальною (Last Stone Weight II).
- **04 (countPartitions)**: Підрахуйте кількість унікальних підмножин nums, сума елементів яких дорівнює target.
- **05 (canPartitionKSubsets)**: Визначте, чи можна розділити масив на k підмножин з однаковою сумою кожної (Partition to K Equal Sum Subsets).
