# Easy_06_CountingBits — Довідка

## Техніка: Порозрядні операції та DP (Bit Manipulation DP)

**Суть:** Поєднання бітових операцій з динамічним програмуванням, де кількість одиничних бітів числа i обчислюється на основі вже розрахованого числа зсунутого вправо (i >> 1) або з маскованим останнім бітом (i & (i-1)).

### Концепт
```java
// dp[i] = dp[i >> 1] + (i & 1)
int[] dp = new int[n + 1];
for (int i = 1; i <= n; i++) {
    dp[i] = dp[i >> 1] + (i & 1);
}
```

**Коли застосовувати:** При обчисленні бітових властивостей послідовності чисел, де структура бітів повторюється з певним кроком/зсувом.

**Складність:** Часова: O(N) — один прохід по всіх числах. Просторова: O(N) для вихідного масиву.

## Вправи (Drills)
- **01 (countOnes)**: Підрахуйте кількість одиничних бітів в одному конкретному числі n за O(K) кроків, де K — кількість встановлених бітів.
- **02 (countBitsBase)**: Реалізуйте класичний алгоритм Counting Bits за допомогою зсуву i >> 1 та перевірки молодшого біта.
- **03 (countBitsLastBit)**: Реалізуйте Counting Bits за допомогою переходу dp[i] = dp[i & (i-1)] + 1.
- **04 (countTotalBits)**: Знайдіть сумарну кількість одиничних бітів у двійковому представленні всіх чисел від 0 до n.
- **05 (countBitsBase3)**: Створіть масив, де для кожного числа i від 0 до n записана кількість цифр '1' у його представленні в трійковій системі числення (base 3) за допомогою DP.
