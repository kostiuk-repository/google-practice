# Medium_03_LongestCommonSubsequence — Довідка

## Техніка: Двовимірне DP зіставлення рядків (LCS DP)

**Суть:** Порівняння двох рядків та знаходження довжини найбільшої спільної підпослідовності. Матриця dp[i][j] зберігає довжину LCS для префіксів рядків довжиною i та j.

### Концепт
```java
// dp[i][j] = (s1[i-1] == s2[j-1]) ? dp[i-1][j-1]+1 : max(dp[i-1][j], dp[i][j-1])
int[][] dp = new int[m + 1][n + 1];
for (int i = 1; i <= m; i++) {
    for (int j = 1; j <= n; j++) {
        if (text1.charAt(i - 1) == text2.charAt(j - 1)) {
            dp[i][j] = dp[i - 1][j - 1] + 1;
        } else {
            dp[i][j] = Math.max(dp[i - 1][j], dp[i][j - 1]);
        }
    }
}
```

**Коли застосовувати:** При зіставленні послідовностей, знаходженні спільного контенту, порівнянні файлів (diff) або аналізі ДНК.

**Складність:** Часова: O(M * N) — заповнення таблиці. Просторова: O(M * N) для матриці або O(min(M, N)) при оптимізації пам'яті до одного-двох рядків.

- **05 (shortestCommonSupersequence)**: Знайдіть довжину найкоротшого рядка, який містить обидва вхідні рядки як підпослідовності (Shortest Common Supersequence).
