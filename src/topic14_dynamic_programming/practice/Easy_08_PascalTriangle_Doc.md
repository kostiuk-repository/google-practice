# Easy_08_PascalTriangle — Довідка

## Техніка: Двовимірна комбінаторна таблиця (Pascal's DP)

**Суть:** Заповнення двовимірного масиву (або списку списків) відповідно до правила додавання сусідніх елементів попереднього рядка.

### Концепт
```java
// row[j] = prevRow[j-1] + prevRow[j]
List<List<Integer>> triangle = new ArrayList<>();
// ... ініціалізація першого рядка
for (int i = 1; i < numRows; i++) {
    List<Integer> row = new ArrayList<>();
    List<Integer> prev = triangle.get(i-1);
    row.add(1);
    for (int j = 1; j < i; j++) {
        row.add(prev.get(j-1) + prev.get(j));
    }
    row.add(1);
    triangle.add(row);
}
```

**Коли застосовувати:** При обчисленні комбінаторних коефіцієнтів (C(n, k)) або побудові сітки додавання.

**Складність:** Часова: O(N^2) — кількість комірок у трикутнику. Просторова: O(N^2) для збереження всього трикутника.

## Вправи (Drills)
- **01 (getRow)**: Отримайте тільки rowIndex-й рядок трикутника Паскаля (0-індексований).
- **02 (generateBase)**: Побудуйте перші numRows рядків трикутника Паскаля.
- **03 (getPascalValue)**: Знайдіть конкретне значення в трикутнику Паскаля на перетині row та col за допомогою рекурсії з мемоізацією.
- **04 (getRowSpaceOptimized)**: Поверніть rowIndex-й рядок трикутника Паскаля, використовуючи лише O(K) додаткової пам'яті.
- **05 (generatePascalMod)**: Згенеруйте перші numRows рядків трикутника Паскаля, де кожне значення обчислюється за модулем mod.
