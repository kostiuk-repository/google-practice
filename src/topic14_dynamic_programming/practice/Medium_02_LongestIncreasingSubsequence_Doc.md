# Medium_02_LongestIncreasingSubsequence — Довідка

## Техніка: DP з бінарним пошуком / Квадратичне DP (LIS DP)

**Суть:** Пошук найдовшої зростаючої підпослідовності. Класичне DP розв'язує задачу за O(N^2), де dp[i] — довжина LIS, що закінчується на nums[i]. Оптимальний підхід використовує масив tails, де tails[i] зберігає мінімальне кінцеве значення LIS довжини i+1. tails заповнюється з бінарним пошуком за O(N log N).

### Концепт
```java
// O(N log N) алгоритм LIS
int[] tails = new int[nums.length];
int size = 0;
for (int x : nums) {
    int i = 0, j = size;
    while (i != j) {
        int m = (i + j) / 2;
        if (tails[m] < x) i = m + 1;
        else j = m;
    }
    tails[i] = x;
    if (i == size) size++;
}
return size;
```

**Коли застосовувати:** При пошуку найдовшої впорядкованої за зростанням/спаданням підмножини елементів, елементи якої не обов'язково є сусідніми.

**Складність:** Часова: O(N^2) для базового DP або O(N log N) з бінарним пошуком. Просторова: O(N) додаткової пам'яті.
