# Medium_01_HouseRobber — Довідка

## Техніка: DP на масивах з вибором стану (State Selection DP)

**Суть:** Визначення оптимального вибору (брати/не брати поточний елемент) за рахунок переходу від станів, які задовольняють обмеження задачі.

### Концепт
```java
// dp[i] = max(dp[i-1], dp[i-2] + nums[i])
int prev1 = 0, prev2 = 0;
for (int num : nums) {
    int curr = Math.max(prev1, prev2 + num);
    prev2 = prev1;
    prev1 = curr;
}
```

**Коли застосовувати:** Коли вибір елемента блокує вибір інших зв'язаних з ним елементів (сусідніх у масиві, нащадків у дереві тощо).

**Складність:** Часова: O(N) — лінійний час. Просторова: O(N) або O(1) при оптимізації змінними.

- **05 (robWithCoolDown)**: Після пограбування будинку ви не можете грабувати наступний будинок наступного дня (кулдаун 1 крок).
