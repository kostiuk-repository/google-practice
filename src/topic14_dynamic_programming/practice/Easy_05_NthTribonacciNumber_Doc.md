# Easy_05_NthTribonacciNumber — Довідка

## Техніка: Послідовності вищих порядків (Tribonacci DP)

**Суть:** Поширення лінійного DP на випадок, коли стан залежить від трьох безпосередньо попередніх станів.

### Концепт
```java
// dp[i] = dp[i-1] + dp[i-2] + dp[i-3]
int a = 0, b = 1, c = 1;
for (int i = 3; i <= n; i++) {
    int d = a + b + c;
    a = b;
    b = c;
    c = d;
}
```

**Коли застосовувати:** Коли крок обчислення вимагає інформації про ширше вікно передісторії (3 і більше попередніх елементів).

**Складність:** Часова: O(N) — лінійний обхід. Просторова: O(N) для збереження або O(1) для трьох змінних.

## Вправи (Drills)
- **01 (tribonacciBase)**: Обчисліть n-те число Трибоначчі за допомогою чистої рекурсії з мемоізацією.
- **02 (tribonacciTab)**: Реалізуйте обчислення n-го числа Трибоначчі за допомогою класичної табуляції (bottom-up).
- **03 (tribonacciSpace)**: Оптимізуйте просторову складність трибоначчі до O(1) додаткової пам'яті.
- **04 (tribonacciK)**: Узагальніть алгоритм для k-боначчі, де кожен елемент є сумою k попередніх. Перші k елементів мають значення: T_0..T_{k-2} = 0, T_{k-1} = 1.
- **05 (tribonacciSum)**: Знайдіть суму перших n чисел Трибоначчі (від T_0 до T_n).
