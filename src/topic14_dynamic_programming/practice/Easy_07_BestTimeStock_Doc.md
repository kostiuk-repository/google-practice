# Easy_07_BestTimeStock — Довідка

## Техніка: DP одного проходу (Kadane's-like DP / State tracking)

**Суть:** Відслідковування та оновлення глобального стану (мінімальна ціна покупки) на кожному кроці і розрахунок потенційного прибутку у поточний день.

### Концепт
```java
int minPrice = Integer.MAX_VALUE;
int maxProfit = 0;
for (int price : prices) {
    minPrice = Math.min(minPrice, price);
    maxProfit = Math.max(maxProfit, price - minPrice);
}
return maxProfit;
```

**Коли застосовувати:** Коли потрібно знайти максимальну різницю між двома елементами масиву, де елемент з більшим індексом віднімає елемент з меншим.

**Складність:** Часова: O(N) — один прохід по масиву. Просторова: O(1) додаткової пам'яті.

## Вправи (Drills)
- **01 (maxProfitBrute)**: Знайдіть максимальний прибуток за допомогою повного перебору всіх можливих пар днів купівлі-продажу.
- **02 (maxProfitOnePass)**: Реалізуйте оптимальне рішення за один прохід.
- **03 (maxProfitMultipleTx)**: Дозволено купувати та продавати акції стільки разів, скільки хочете (але тільки одну транзакцію одночасно). Знайдіть максимальний прибуток (Best Time to Buy and Sell Stock II).
- **04 (maxProfitWithCooldown)**: Те саме, що й багаторазові транзакції, але після продажу акції ви не можете купувати її наступного дня (кулдаун 1 день).
- **05 (maxProfitWithFee)**: Те саме, що й багаторазові транзакції, але за кожну завершену транзакцію стягується фіксована комісія fee.
