# Easy_03_MinCostClimbingStairs — Довідка

## Техніка: Динамічне програмування на масивах з мінімізацією вартості

**Суть:** Пошук оптимального шляху найменшої вартості через вибір мінімуму між альтернативними попередніми станами на кожному кроці.

### Концепт
```java
// dp[i] = cost[i] + min(dp[i-1], dp[i-2])
int first = cost[0];
int second = cost[1];
for (int i = 2; i < n; i++) {
    int curr = cost[i] + Math.min(first, second);
    first = second;
    second = curr;
}
return Math.min(first, second);
```

**Коли застосовувати:** При знаходженні мінімальної вартості переходу до цільового стану, коли кожен крок додає певну ціну.

**Складність:** Часова: O(N) — лінійний обхід масиву вартостей. Просторова: O(N) для таблиці або O(1) за допомогою двох змінних.
