# Easy_10_HouseRobberEasy — Довідка

## Техніка: Одновимірне DP на незалежних виборах (House Robber DP)

**Суть:** Знаходження максимуму при виборі несусідніх елементів масиву. Стан dp[i] представляє максимальний виграш до i-го будинку включно.

### Концепт
```java
// dp[i] = max(dp[i-1], dp[i-2] + nums[i])
int prev2 = 0;
int prev1 = 0;
for (int num : nums) {
    int tmp = prev1;
    prev1 = Math.max(prev1, prev2 + num);
    prev2 = tmp;
}
return prev1;
```

**Коли застосовувати:** При виборі підмножини елементів масиву з умовою, що жодні два обрані елементи не можуть бути сусідніми.

**Складність:** Часова: O(N) — один обхід масиву. Просторова: O(N) для збереження або O(1) при використанні двох змінних.

## Вправи (Drills)
- **01 (robBase)**: Знайдіть максимальну суму грошей, яку можна вкрасти, за допомогою масиву DP.
- **02 (robSpace)**: Реалізуйте алгоритм House Robber з просторовою складністю O(1).
- **03 (robCircular)**: Будинки розташовані по колу (перший є сусідом останнього). Знайдіть максимальну суму.
- **04 (robSkipTwo)**: Те саме, але після пограбування будинку ви повинні пропустити щонайменше 2 наступні будинки (заборонено грабувати сусідні та через один).
- **05 (robPath)**: Поверніть масив індексів будинків, які потрібно пограбувати для отримання максимальної суми.
