# Easy_04_IntersectionTwoArrays — Довідка

## Техніка: HashSet Membership Across Two Collections

**Суть:** заносимо всі елементи одного масиву в `HashSet`, потім
проходимо другий масив і для кожного елемента перевіряємо належність
до множини за O(1). Це дозволяє порівнювати дві колекції без
вкладеного циклу O(N*M).

```java
Set<Integer> set1 = new HashSet<>();
for (int x : nums1) set1.add(x);

Set<Integer> result = new HashSet<>();
for (int x : nums2) {
    if (set1.contains(x)) {
        result.add(x);
    }
}
```

**Коли застосовувати:** будь-яка задача "перетин", "об'єднання",
"різниця" двох чи більше колекцій — перевірка спільних/унікальних
елементів без сортування.

**Складність:** O(N + M) час, O(N) додаткової пам'яті для першої
множини.
