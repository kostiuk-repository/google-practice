# Easy_05_HappyNumber — Довідка

## Техніка: HashSet для детекції циклу в ітеративному процесі

**Суть:** застосовуємо детерміновану трансформацію (сума квадратів
цифр) знову і знову. Якщо число колись стає 1 — воно "щасливе". Якщо
процес зациклюється, не досягнувши 1, то рано чи пізно ми побачимо
значення, яке вже було. `HashSet` зберігає всі побачені значення;
щойно нове значення вже є в множині — це підтверджений нескінченний
цикл без 1.

```java
Set<Integer> seen = new HashSet<>();
int current = n;
while (current != 1 && !seen.contains(current)) {
    seen.add(current);
    current = sumOfSquaredDigits(current);
}
return current == 1;
```

**Коли застосовувати:** будь-який детермінований ітеративний процес
("наступний стан однозначно визначається поточним"), де треба
визначити, чи він завершується у певному стані, чи зациклюється —
виявлення циклів у функціональних графах, послідовностях Коллаца-типу.

**Складність:** O(K) час і O(K) пам'ять, де K — кількість кроків до
циклу/до 1 (на практиці мала стала кількість для чисел даного
діапазону). Без HashSet (Флойд) — O(K) час, O(1) пам'ять.
