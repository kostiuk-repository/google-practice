# Easy_02_ContainsDuplicate — Довідка

## Техніка: HashSet Membership Check

**Суть:** проходимо масив і для кожного елемента перевіряємо, чи він
уже є в `HashSet`. Якщо є — знайдено дублікат; якщо ні — додаємо його
до множини і йдемо далі. `HashSet` дає O(1) середню перевірку
належності, тому весь прохід лінійний.

```java
Set<Integer> seen = new HashSet<>();
for (int x : nums) {
    if (!seen.add(x)) { // add() повертає false, якщо елемент вже був
        return true;
    }
}
return false;
```

**Коли застосовувати:** будь-яка задача "чи є повторення", "скільки
разів щось повторюється", "чи перетинаються дві колекції" — базова
перевірка належності є будівельним блоком майже для всіх задач на
хеш-таблиці.

**Складність:** O(N) час, O(N) пам'яті у гіршому випадку (усі елементи
різні і потрапляють у множину).
