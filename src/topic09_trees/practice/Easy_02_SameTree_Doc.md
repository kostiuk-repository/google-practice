# Easy_02_SameTree — Довідка

## Техніка: Synchronized Two-Tree Recursion

**Суть:** обходимо ОДРАЗУ два дерева паралельно, одним рекурсивним
викликом на пару вузлів. Базові випадки: обидва null → true/рівні;
рівно один null → false/нерівні; інакше порівнюємо значення поточної
пари і рекурсивно перевіряємо пари дітей.

```java
boolean isSameTree(TreeNode p, TreeNode q) {
    if (p == null && q == null) return true;
    if (p == null || q == null) return false;
    return p.val == q.val
        && isSameTree(p.left, q.left)
        && isSameTree(p.right, q.right);
}
```

**Коли застосовувати:** будь-яке порівняння двох дерев — на рівність,
на дзеркальність, на структурну схожість, на "піддерево одного є копією
іншого". Ключова ознака: два дерева обходяться СИНХРОННО, пара за парою,
а не по одному.

**Складність:** O(min(N, M)) час у гіршому випадку (обхід припиняється
на першій розбіжності), O(min(H1, H2)) пам'яті на стек рекурсії.

## Drills

1. **Easy_02_01** — порівняння лише структури (форми), значення ігноруються.
2. **Easy_02_02** — перевірка дзеркальності: p.left ↔ q.right, p.right ↔ q.left.
3. **Easy_02_03** — підрахунок кількості позицій зі збігом замість boolean.
4. **Easy_02_04** — порівняння значень з допуском (tolerance) замість строгої рівності.
5. **Easy_02_05** — та сама перевірка, але ІТЕРАТИВНО через стек пар вузлів.
