# Easy_01_BinarySearch — Довідка

## Техніка: Класичний Binary Search (звуження lo/hi)

**Суть:** два вказівники `left` і `right` окреслюють поточний діапазон
пошуку. На кожному кроці обчислюємо `mid` і порівнюємо `nums[mid]` із
`target`: якщо рівні — знайшли; якщо `nums[mid] < target` — відкидаємо
ліву половину (`left = mid + 1`); інакше відкидаємо праву половину
(`right = mid - 1`). Діапазон звужується вдвічі щокроку.

```java
int left = 0, right = nums.length - 1;
while (left <= right) {
    int mid = left + (right - left) / 2;
    if (nums[mid] == target) return mid;
    else if (nums[mid] < target) left = mid + 1;
    else right = mid - 1;
}
return -1;
```

**Коли застосовувати:** масив (або будь-яка індексована послідовність)
відсортований, і потрібно знайти конкретне значення або довести його
відсутність за O(log n) замість лінійного проходу.

**Складність:** O(log N) час; O(1) пам'яті для ітеративної версії,
O(log N) для рекурсивної (стек викликів).
