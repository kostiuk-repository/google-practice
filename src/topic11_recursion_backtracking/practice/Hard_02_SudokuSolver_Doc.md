# Hard_02_SudokuSolver — Довідка

## Техніка: Constraint Propagation / Search Space Backtracking

**Суть:** Заповнення клітинок таблиці з дотриманням умов Sudoku. Для кожної порожньої клітинки випробовуються всі можливі значення від 1 до 9. Якщо вибір призводить до глухого кута, значення стирається і відбувається відкат.

```java
boolean solve(char[][] board) {
    for (int i = 0; i < 9; i++) {
        for (int j = 0; j < 9; j++) {
            if (board[i][j] == '.') {
                for (char c = '1'; c <= '9'; c++) {
                    if (isValid(board, i, j, c)) {
                        board[i][j] = c;
                        if (solve(board)) return true;
                        board[i][j] = '.';
                      }
                  }
                  return false;
              }
          }
      }
      return true;
}
```

**Коли застосовувати:** Повне заповнення обмеженого простору станів (наприклад, головоломки, кросворди), де кожен крок звужує вибір для наступних клітинок.

**Складність:** O(9^(R*C)) час, O(R*C) пам'ять.

## Drills

1. **Hard_02_01** — Sudoku Validator: Визначте, чи є заповнена частина таблиці 9x9 Sudoku правильною (перевірка без розв'язання).
2. **Hard_02_02** — Sudoku Solver Count: Рекурсивно порахуйте кількість можливих правильних розв'язків для частково заповненої дошки Sudoku.
3. **Hard_02_03** — 4x4 Mini Sudoku Solver: Розв'яжіть міні-Sudoku розміром 4x4 з числами 1-4 та субгридами 2x2 in-place.
4. **Hard_02_04** — Sudoku Solver with Diagonal Constraints: Розв'яжіть Sudoku 9x9, де обидві головні діагоналі також мають містити унікальні цифри 1-9.
5. **Hard_02_05** — Sudoku Solver Lexicographical: Розв'яжіть дошку Sudoku. Якщо рішень декілька, переконайтеся, що дошка заповнена лексикографічно найменшим розв'язком.

Розв'язуй у порядку 01 → 05, не підглядаючи в Hard_02_SudokuSolver.java.
