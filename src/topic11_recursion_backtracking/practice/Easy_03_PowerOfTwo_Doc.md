# Easy_03_PowerOfTwo — Довідка

## Техніка: Recursive Decrease-and-Conquer

**Суть:** Задача вирішується шляхом послідовного зменшення аргументу (наприклад, ділення на основу). Базовий випадок — коли число стає рівним 1 (істина) або не ділиться без остачі / менше 1 (хибність).

```java
boolean isPowerOfTwo(int n) {
    if (n <= 0) return false;
    if (n == 1) return true;
    return (n % 2 == 0) && isPowerOfTwo(n / 2);
}
```

**Коли застосовувати:** Коли потрібно перевірити приналежність числа до певної математичної послідовності степенів або виконати рекурсивні побітові операції.

**Складність:** O(log n) час та пам'ять.

## Drills

1. **Easy_03_01** — Is Power of Four?: Дано ціле число n. Поверніть true, якщо воно є степенем 4, інакше false.
2. **Easy_03_02** — Is Power of K?: Дано два числа n та k (k > 1). Поверніть true, якщо n є степенем k, інакше false.
3. **Easy_03_03** — Count Set Bits: Рекурсивно обчисліть кількість встановлених бітів (вагу Хеммінга) для цілого числа n.
4. **Easy_03_04** — Recursive Log Base 2: Дано додатне ціле число n. Рекурсивно знайдіть floor(log2(n)).
5. **Easy_03_05** — Next Power of Two: Дано ціле число n. Рекурсивно знайдіть найменший степінь двійки, який є більшим або рівним n.

Розв'язуй у порядку 01 → 05, не підглядаючи в Easy_03_PowerOfTwo.java.
