# Easy_05_MergeTwoListsRecursive — Довідка

## Техніка: Recursive List Merging / Restructuring

**Суть:** Робота з однозв'язними списками через рекурсію. Замість ітеративного циклу порівнюються перші елементи списків, і результат злиття решти списку рекурсивно присвоюється полю `next` вибраного вузла.

```java
ListNode merge(ListNode l1, ListNode l2) {
    if (l1 == null) return l2;
    if (l2 == null) return l1;
    if (l1.val <= l2.val) {
        l1.next = merge(l1.next, l2);
        return l1;
    } else {
        l2.next = merge(l1, l2.next);
        return l2;
    }
}
```

**Коли застосовувати:** Злиття, розділення, реверсування чи модифікація зв'язаних списків без використання додаткових ітераторів.

**Складність:** O(n + m) час і пам'ять.

## Drills

1. **Easy_05_01** — Merge Two Sorted Lists Descending: Рекурсивно об'єднайте два відсортовані за зростанням списки в один список, відсортований за спаданням.
2. **Easy_05_02** — Interleave Two Lists: Рекурсивно перемежуйте вузли двох списків l1 та l2.
3. **Easy_05_03** — Merge N Sorted Lists: Дано список відсортованих зв'язаних списків. Об'єднайте їх рекурсивно за принципом 'розділяй і володарюй'.
4. **Easy_05_04** — Delete Duplicates from Sorted List: Дано відсортований зв'язаний список. Рекурсивно видаліть усі дублікати, щоб кожен елемент зустрічався лише один раз.
5. **Easy_05_05** — Split Sorted List into Odd and Even Nodes: Рекурсивно розділіть список на вузли з непарними та парними індексами й поверніть їх як масив із двох вузлів [oddHead, evenHead].

Розв'язуй у порядку 01 → 05, не підглядаючи в Easy_05_MergeTwoListsRecursive.java.
