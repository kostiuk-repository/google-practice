# Easy_04_ReverseStringRecursive — Довідка

## Техніка: Two-pointer Recursion

**Суть:** Рекурсивне проходження масиву або рядка з двома вказівниками (початок і кінець), які рухаються назустріч один одному. На кожному кроці міняються місцями елементи на позиціях вказівників, після чого виконується рекурсивний виклик для звуженого діапазону.

```java
void reverse(char[] s, int left, int right) {
    if (left >= right) return;
    char temp = s[left];
    s[left] = s[right];
    s[right] = temp;
    reverse(s, left + 1, right - 1);
}
```

**Коли застосовувати:** Модифікація масивів/рядків з обох кінців, перевірка паліндромів, або дзеркальне відображення без додаткової пам'яті.

**Складність:** O(n) час, O(n) пам'ять для стека викликів.

3. **Easy_04_03** — Is Palindrome: Рекурсивно перевірте, чи є масив символів паліндромом від індексу 'left' до 'right'.
4. **Easy_04_04** — Swap Pairs: Рекурсивно поміняйте місцями сусідні пари символів in-place починаючи з заданого індексу.
5. **Easy_04_05** — Rotate Array: Розгорніть масив символів праворуч на k кроків in-place за допомогою рекурсивних реверсів.
