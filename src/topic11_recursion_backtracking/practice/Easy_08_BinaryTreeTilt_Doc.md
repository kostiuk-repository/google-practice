# Easy_08_BinaryTreeTilt — Довідка

## Техніка: Recursive Post-Order Tree Accumulation

**Суть:** Обхід бінарного дерева у зворотному порядку (Post-Order: ліве, праве, корінь). Під час обходу кожний вузол обчислює суму значень своїх піддерев і передає її вгору, одночасно оновлюючи глобальну чи накопичувальну змінну на основі різниці (tilt).

```java
int tilt = 0;
int sum(TreeNode root) {
    if (root == null) return 0;
    int left = sum(root.left);
    int right = sum(root.right);
    tilt += Math.abs(left - right);
    return root.val + left + right;
}
```

**Коли застосовувати:** Коли для обчислення метрики вузла потрібна інформація від обох його дітей (наприклад, висота, сума піддерева, баланс).

**Складність:** O(n) час, O(h) пам'ять.

## Drills

1. **Easy_08_01** — Height of Binary Tree: Рекурсивно обчисліть максимальну глибину (висоту) бінарного дерева.
2. **Easy_08_02** — Balance Factor of Binary Tree: Рекурсивно перевірте, чи є бінарне дерево збалансованим за висотою.
3. **Easy_08_03** — Maximum Path Sum from Root to Leaf: Рекурсивно знайдіть максимальну суму значень уздовж будь-якого шляху від кореня до листка.
4. **Easy_08_04** — Diameter of Binary Tree: Рекурсивно обчисліть діаметр бінарного дерева (найдовший шлях між двома будь-якими вузлами).
5. **Easy_08_05** — Sum of Equal Subtree Nodes: Знайдіть суму всіх значень вузлів, у яких ліве та праве піддерева мають однакову суму значень.

Розв'язуй у порядку 01 → 05, не підглядаючи в Easy_08_BinaryTreeTilt.java.
