# Easy_07_PowerOfThree — Довідка

## Техніка: Recursive Base Division

**Суть:** Перевірка властивості степеня аналогічно основі 2, але з діленням на 3. Альтернативно, це може бути представлено у вигляді переводу в трійкову систему чи рекурсивного розбиття.

```java
boolean isPowerOfThree(int n) {
    if (n <= 0) return false;
    if (n == 1) return true;
    return (n % 3 == 0) && isPowerOfThree(n / 3);
}
```

**Коли застосовувати:** Задачі з перевірки подільності, перетворення основ чи специфічних числових властивостей.

**Складність:** O(log_3 n) час та пам'ять.

## Drills

1. **Easy_07_01** — Count Divisions By Three: Рекурсивно знайдіть, скільки разів ціле число n можна поділити на 3 без остачі.
2. **Easy_07_02** — Is Power of Three or Five: Поверніть true, якщо n є степенем 3 або степенем 5, рекурсивно.
3. **Easy_07_03** — Is Ternary Palindrome: Рекурсивно перевірте, чи є представлення цілого числа n у трійковій системі числення паліндромом.
4. **Easy_07_04** — Largest Power of Three Less Than N: Рекурсивно знайдіть найбільший степінь трійки, який є строго меншим за n.
5. **Easy_07_05** — Base 3 Representation: Рекурсивно перетворіть ціле число n на його представлення у трійковій системі як рядок.

Розв'язуй у порядку 01 → 05, не підглядаючи в Easy_07_PowerOfThree.java.
