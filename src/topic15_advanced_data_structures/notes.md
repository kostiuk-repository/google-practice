# Advanced Data Structures - Study Notes

Ця тема присвячена структурам даних для оптимізації специфічних запитів:
* **Trie:** Оптимізація пошуку префіксів у рядках.
* **Segment Tree / Fenwick Tree:** Запити на діапазонах (Range queries) з оновленням за $O(\log N)$.
* **Monotonic Stack / Queue:** Пошук наступного більшого/меншого елемента за амортизований $O(1)$ час.
