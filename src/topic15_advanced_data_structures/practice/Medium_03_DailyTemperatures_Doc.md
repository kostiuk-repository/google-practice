# Medium_03_DailyTemperatures — Довідка

## Техніка: Монотонний стек (Daily Temperatures)

**Суть:** Оптимальне знаходження наступного теплішого дня за допомогою збереження індексів у стеці, що підтримує спадний порядок температур. Кожен індекс додається в стек один раз, що забезпечує лінійну складність O(N).

```java
int[] result = new int[temperatures.length];
Stack<Integer> stack = new Stack<>();
for (int i = 0; i < temperatures.length; i++) {
    while (!stack.isEmpty() && temperatures[i] > temperatures[stack.peek()]) {
        int prev = stack.pop();
        result[prev] = i - prev;
    }
    stack.push(i);
}
return result;
```

**Коли застосовувати:** Пошук найближчих елементів з більшим значенням для кожної позиції в масиві.

**Складність:** O(N) час, O(N) додаткова пам'ять для стека.
