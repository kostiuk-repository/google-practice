# Easy_10_MonotonicQueueIntro — Довідка

## Техніка: Монотонна черга (Monotonic Deque Introduction)

**Суть:** Введення в використання Deque для збереження елементів у монотонному порядку. У монотонно спадній черзі нові елементи витісняють усі менші елементи з кінця черги перед додаванням. Це дає швидкий O(1) доступ до максимуму на початку черги.

```java
Deque<Integer> dq = new LinkedList<>();
// push(x): 
while(!dq.isEmpty() && dq.peekLast() < x) dq.pollLast();
dq.addLast(x);
// max: dq.peekFirst();
```

**Коли застосовувати:** Задачі ковзного вікна, де потрібно швидко знаходити екстремуми (мінімум/максимум) при переміщенні вікна.

**Складність:** Амортизована складність додавання/видалення елемента — O(1), отримання максимуму — O(1).
