# Easy_06_ImplementTrieEasy — Довідка

## Техніка: Префіксне дерево (Trie Node array-based)

**Суть:** Побудова Trie за допомогою вузлів, які містять масив дочірніх посилань фіксованого розміру (зазвичай 26 для малих латинських літер). Це забезпечує швидкість переходу O(1) за символ.

```java
class TrieNode {
    TrieNode[] children = new TrieNode[26];
    boolean isEndOfWord;
}
```

**Коли застосовувати:** Швидкий пошук за префіксом, автодоповнення, словникові задачі.

**Складність:** Пошук/вставка: O(L) за час, де L — довжина слова. Пам'ять: O(W * L * 26) в найгіршому випадку.

## Drills

   1. **Easy_06_01** — Перетворення символу 'a'-'z' в індекс 0-25.
   2. **Easy_06_02** — Створення структури TrieNode та перевірка наявності дочірнього вузла.
   3. **Easy_06_03** — Вставка слова в TrieNode.
   4. **Easy_06_04** — Пошук слова в TrieNode.
   5. **Easy_06_05** — Пошук префікса в TrieNode.

Розв'язуй у порядку 01 → 05, не підглядаючи в Easy_06_ImplementTrieEasy.java.
