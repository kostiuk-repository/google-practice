# Easy_05_RangeSumMutableEasy — Довідка

## Техніка: Оновлення масиву та інтервальні запити (Mutable Range Sum)

**Суть:** Проста модель оновлення значень у масиві. На відміну від незмінного масиву, тут при змінах наївне оновлення працює за O(1), але сума діапазону виконується за O(N). Це підводить до необхідності використання дерева сегментів.

```java
// Наївна реалізація
public void update(int[] nums, int index, int val) {
    nums[index] = val;
}
public int sumRange(int[] nums, int left, int right) {
    int sum = 0;
    for (int i = left; i <= right; i++) sum += nums[i];
    return sum;
}
```

**Коли застосовувати:** Коли масив часто змінюється і потрібно періодично рахувати суму елементів на проміжках.

**Складність:** update: O(1), sumRange: O(N) (для наївної версії).
