# Easy_04_LongestCommonPrefix — Довідка

## Техніка: Найдовший спільний префікс (Longest Common Prefix)

**Суть:** Пошук найбільшої початкової підпослідовності символів, спільної для всіх рядків у масиві. Можна розв'язати горизонтальним скануванням, вертикальним скануванням, сортуванням або бінарним пошуком.

```java
if (strs == null || strs.length == 0) return "";
String prefix = strs[0];
for (int i = 1; i < strs.length; i++) {
    while (strs[i].indexOf(prefix) != 0) {
        prefix = prefix.substring(0, prefix.length() - 1);
        if (prefix.isEmpty()) return "";
    }
}
return prefix;
```

**Коли застосовувати:** Аналіз схожості групи текстових рядків, автодоповнення, префіксне стиснення.

**Складність:** O(S) час, де S — сумарна довжина всіх символів у рядках; O(1) додаткова пам'ять.
