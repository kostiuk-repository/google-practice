# Medium_04_DesignAddSearchWords — Довідка

## Техніка: Trie з рекурсивним пошуком та регулярними виразами (Wildcard Search)

**Суть:** Пошук слів у префіксному дереві з підтримкою символу підстановки '.'. Коли в пошуковому рядку зустрічається '.', алгоритм має перевірити рекурсивно всі можливі переходи до дочірніх нод на даному рівні дерева за допомогою DFS.

```java
boolean search(String word, int index, TrieNode node) {
    if (node == null) return false;
    if (index == word.length()) return node.isEndOfWord;
    char c = word.charAt(index);
    if (c == '.') {
        for (TrieNode child : node.children) {
            if (child != null && search(word, index + 1, child)) return true;
        }
        return false;
    } else {
        return search(word, index + 1, node.children[c - 'a']);
    }
}
```

**Коли застосовувати:** Пошук слів за маскою або шаблоном, розпізнавання регулярних виразів у словниках.

**Складність:** Вставка: O(L), Пошук: O(26^N) у найгіршому випадку, коли всі символи є крапками, де N — кількість символів підстановки.
