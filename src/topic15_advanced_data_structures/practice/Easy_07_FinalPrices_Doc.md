# Easy_07_FinalPrices — Довідка

## Техніка: Монотонний стек (Next Smaller Element)

**Суть:** Монотонний стек підтримує елементи в порядку зростання/спадання. Для пошуку наступного меншого або рівного елемента ми зберігаємо індекси елементів у стеку та виштовхуємо їх, коли зустрічаємо менший елемент.

```java
Stack<Integer> stack = new Stack<>();
for (int i = 0; i < prices.length; i++) {
    while (!stack.isEmpty() && prices[stack.peek()] >= prices[i]) {
        int idx = stack.pop();
        prices[idx] -= prices[i];
    }
    stack.push(i);
}
```

**Коли застосовувати:** Пошук найближчих менших/більших елементів праворуч чи ліворуч у масиві за лінійний час.

**Складність:** O(N) час, оскільки кожен елемент додається та виштовхується зі стеку щонайбільше один раз; O(N) додаткова пам'ять.
