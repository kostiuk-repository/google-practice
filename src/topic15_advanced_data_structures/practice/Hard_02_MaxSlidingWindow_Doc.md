# Hard_02_MaxSlidingWindow — Довідка

## Техніка: Sliding Window Maximum (Максимум у ковзному вікні)

**Суть:** Пошук максимальних елементів для кожного підмасиву довжини k при зсуві вікна. За допомогою двобічної черги (Deque), яка зберігає індекси елементів у монотонно спадному порядку за значеннями, ми можемо знаходити максимум кожного вікна за O(1) амортизовано.

```java
Deque<Integer> q = new LinkedList<>();
for (int i = 0; i < nums.length; i++) {
    if (!q.isEmpty() && q.peek() < i - k + 1) q.poll();
    while (!q.isEmpty() && nums[q.peekLast()] < nums[i]) q.pollLast();
    q.offer(i);
    if (i >= k - 1) result[r++] = nums[q.peek()];
}
```

**Коли застосовувати:** Знаходження локальних максимумів/мінімумів у динамічних діапазонах фіксованої довжини.

**Складність:** O(N) час, O(k) додаткова пам'ять для Deque.
