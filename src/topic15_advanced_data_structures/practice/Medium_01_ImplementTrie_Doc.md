# Medium_01_ImplementTrie — Довідка

## Техніка: Trie (Prefix Tree) з HashMap дочірніх нод

**Суть:** Реалізація префіксного дерева, де замість масиву фіксованого розміру 26 для кожної ноди використовується HashMap<Character, TrieNode>. Це дозволяє динамічно виділяти пам'ять лише для наявних переходів та підтримувати символи Unicode.

```java
class TrieNode {
    Map<Character, TrieNode> children = new HashMap<>();
    boolean isWord;
}
```

**Коли застосовувати:** Коли алфавіт символів є великим або невідомим заздалегідь (наприклад, юнікод, повний ASCII).

**Складність:** Час: O(L) для вставки та пошуку. Пам'ять: O(W * L) — загалом менше витрат пам'яті на порожні посилання порівняно з масивом.
