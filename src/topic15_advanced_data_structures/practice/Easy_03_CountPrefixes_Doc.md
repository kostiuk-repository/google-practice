# Easy_03_CountPrefixes — Довідка

## Техніка: Рядковий префіксний збіг (Prefix matching)

**Суть:** Аналіз та підрахунок рядків, що починаються з певного префікса. Може виконуватися наївно через startsWith() або ефективно для великої кількості запитів за допомогою Trie.

```java
int count = 0;
for (String word : words) {
    if (s.startsWith(word)) {
        count++;
    }
}
```

**Коли застосовувати:** Пошук та фільтрація слів у словнику за заданим початком.

**Складність:** O(N * L) час, де N — кількість слів, L — середня довжина слова; O(1) пам'ять (або O(W * L) для Trie).

## Drills

   1. **Easy_03_01** — Перевірка, чи є рядок prefix префіксом рядка s.
   2. **Easy_03_02** — Підрахунок слів у масиві, які точно дорівнюють рядку s.
   3. **Easy_03_03** — Підрахунок слів у масиві, які починаються на заданий префікс.
   4. **Easy_03_04** — Підрахунок слів з words, які є префіксами s, використовуючи посимвольне порівняння.
   5. **Easy_03_05** — Підрахунок унікальних слів з words, які є префіксами s.

Розв'язуй у порядку 01 → 05, не підглядаючи в Easy_03_CountPrefixes.java.
