# Medium_02_RangeSumQueryMutable — Довідка

## Техніка: 1D Segment Tree / Fenwick Tree (Binary Indexed Tree)

**Суть:** Забезпечення операцій оновлення елементів та запитів суми інтервалу за O(log N) за допомогою дерева сегментів або дерева Фенвіка. Дерево сегментів ділить масив навпіл рекурсивно, а Fenwick Tree використовує бітові операції для визначення батьківських зв'язків.

```java
// Оновлення Fenwick tree
void update(int i, int delta) {
    for (; i < tree.length; i += i & -i) tree[i] += delta;
}
```

**Коли застосовувати:** Часті модифікації елементів масиву в поєднанні з інтервальними запитами.

**Складність:** Оновлення: O(log N), Запит суми: O(log N) за час. Пам'ять: O(N) додатково.

## Drills

   1. **Medium_02_01** — Побудова одновимірного дерева сегментів.
   2. **Medium_02_02** — Запит суми інтервалу [left, right] у 1D дереві сегментів.
   3. **Medium_02_03** — Оновлення значення елемента за індексом у 1D дереві сегментів.
   4. **Medium_02_04** — Додавання значення delta до елемента за індексом у дереві Фенвіка.
   5. **Medium_02_05** — Запит префіксної суми від 1 до index у дереві Фенвіка.

Розв'язуй у порядку 01 → 05, не підглядаючи в Medium_02_RangeSumQueryMutable.java.
