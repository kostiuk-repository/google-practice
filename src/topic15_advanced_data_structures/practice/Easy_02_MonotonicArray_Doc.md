# Easy_02_MonotonicArray — Довідка

## Техніка: Monotonic Array Check (Однопрохідний аналіз)

**Суть:** Перевірка, чи є масив повністю незростаючим або неспадним за один прохід шляхом порівняння сусідніх елементів nums[i] та nums[i-1].

```java
boolean increasing = true, decreasing = true;
for (int i = 1; i < nums.length; i++) {
    if (nums[i] > nums[i - 1]) decreasing = false;
    if (nums[i] < nums[i - 1]) increasing = false;
}
return increasing || decreasing;
```

**Коли застосовувати:** Коли треба перевірити відсортованість/монотонність послідовності без потреби у повній її сортуванні.

**Складність:** O(N) час, O(1) додаткова пам'ять.

## Drills

   1. **Easy_02_01** — Перевірка, чи є масив монотонно неспадним.
   2. **Easy_02_02** — Перевірка, чи є масив монотонно незростаючим.
   3. **Easy_02_03** — Перевірка, чи є масив строго зростаючим або строго спадним (без однакових сусідніх елементів).
   4. **Easy_02_04** — Повернення першого індексу, де порушується монотонність, або -1.
   5. **Easy_02_05** — Перевірка, чи можна зробити масив монотонним, змінивши щонайбільше один елемент.

Розв'язуй у порядку 01 → 05, не підглядаючи в Easy_02_MonotonicArray.java.
