# Easy_01_RangeSumQuery — Довідка

## Техніка: Prefix Sum Array (Immutable)

**Суть:** Попереднє обчислення сум елементів від початку масиву для швидкого знаходження суми будь-якого підмасиву [L, R] за O(1) за формулою: prefix[R + 1] - prefix[L].

```java
int[] prefix = new int[nums.length + 1];
for (int i = 0; i < nums.length; i++) {
    prefix[i + 1] = prefix[i] + nums[i];
}
// sumRange(L, R) = prefix[R + 1] - prefix[L];
```

**Коли застосовувати:** Коли масив не змінюється (immutable), і потрібно дуже часто та швидко отримувати суми на довільних проміжках.

**Складність:** O(N) для передобчислення, O(1) на один запит, O(N) додаткова пам'ять.

## Drills

   1. **Easy_01_01** — Запит суми від початку масиву до індексу `right`.
   2. **Easy_01_02** — Побудова масиву префіксних сум за вихідним масивом (розмір N + 1).
   3. **Easy_01_03** — Обчислення суми в діапазоні [left, right] за допомогою готового префіксного масиву.
   4. **Easy_01_04** — Знаходження максимальної суми префікса в масиві.
   5. **Easy_01_05** — Обробка списку запитів [left, right] на незмінному масиві.

Розв'язуй у порядку 01 → 05, не підглядаючи в Easy_01_RangeSumQuery.java.
