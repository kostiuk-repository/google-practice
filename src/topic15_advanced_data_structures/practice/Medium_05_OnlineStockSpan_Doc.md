# Medium_05_OnlineStockSpan — Довідка

## Техніка: Онлайн монотонний стек (Online Stock Span)

**Суть:** Обчислення інтервалу цін акцій у реальному часі. Для кожної нової ціни ми хочемо знайти кількість послідовних попередніх днів із ціною, меншою або рівною поточній. Ми зберігаємо пари [ціна, інтервал] у спадному стеці та об'єднуємо інтервали.

```java
Stack<int[]> stack = new Stack<>();
public int next(int price) {
    int span = 1;
    while (!stack.isEmpty() && stack.peek()[0] <= price) {
        span += stack.pop()[1];
    }
    stack.push(new int[]{price, span});
    return span;
}
```

**Коли застосовувати:** Аналіз потокових даних (stream of data), де потрібно рахувати тривалість певного тренду.

**Складність:** O(1) амортизований час на кожен запит, O(N) додаткова пам'ять для збереження цін.
