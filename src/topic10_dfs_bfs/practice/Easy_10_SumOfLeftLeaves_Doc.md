# Easy_10_SumOfLeftLeaves — Довідка

## Техніка: Parent-Child Contextual Recursion

**Суть:** підсумувати тільки ті листкові вузли, які є лівими дітьми своїх батьків. Сама по собі перевірка `node.left == null && node.right == null` ідентифікує листок, але контекст "лівий" можна дізнатися лише від батьківського вузла перед переходом до лівого нащадка.

```java
int sumLeft(TreeNode node, boolean isLeft) {
    if (node == null) return 0;
    if (node.left == null && node.right == null) {
        return isLeft ? node.val : 0;
    }
    return sumLeft(node.left, true) + sumLeft(node.right, false);
}
```

**Коли застосовувати:** обчислення характеристик вузлів дерева, де критерій фільтрації залежить від напрямку ребра (лівий/правий нащадок).

**Складність:** O(N) час на повний обхід дерева, O(H) пам'ять для стеку викликів рекурсії.

5. **Easy_10_05** — капстоун: сума лівих листків, які знаходяться на точно заданій глибині від кореня.
