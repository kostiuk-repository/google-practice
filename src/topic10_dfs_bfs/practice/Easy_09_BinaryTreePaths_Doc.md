# Easy_09_BinaryTreePaths — Довідка

## Техніка: DFS Path Backtracking / Path Formulation

**Суть:** згенерувати всі шляхи від кореня до листків. Під час рекурсивного спуску ми будуємо поточний шлях (наприклад, додаючи вузол до StringBuilder або списку), а при поверненні назад (backtracking) прибираємо цей вузол зі стану, щоб інші гілки могли будувати свої шляхи.

```java
void dfs(TreeNode node, String path, List<String> paths) {
    if (node == null) return;
    path += node.val;
    if (node.left == null && node.right == null) {
        paths.add(path);
        return;
    }
    dfs(node.left, path + "->", paths);
    dfs(node.right, path + "->", paths);
}
```

**Коли застосовувати:** генерація всіх шляхів, фільтрація шляхів за сумою або певними умовами, представлення шляхів у вигляді рядків.

**Складність:** O(N * H) час, оскільки кожен вузол відвідується, і на кожному листку копіюється шлях довжиною H. O(H) пам'ять на стек рекурсії та збереження поточного шляху.
