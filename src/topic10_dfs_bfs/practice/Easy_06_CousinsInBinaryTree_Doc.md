# Easy_06_CousinsInBinaryTree — Довідка

## Техніка: DFS/BFS Node Context Tracking (Parent & Depth)

**Суть:** знайти глибину та батьківський вузол для двох заданих значень x та y у бінарному дереві. Якщо вони знаходяться на одній глибині, але мають різних батьків — вони є кузенами. Обхід може бути виконаний як за допомогою DFS, так і BFS, де під час спуску передається/зберігається інформація про батьківський вузол та поточний рівень (глибину).

```java
// Приклад DFS з передачею контексту батька та глибини
void dfs(TreeNode node, TreeNode parent, int depth, int x, int y) {
    if (node == null) return;
    if (node.val == x) {
        xParent = parent;
        xDepth = depth;
    } else if (node.val == y) {
        yParent = parent;
        yDepth = depth;
    }
    dfs(node.left, node, depth + 1, x, y);
    dfs(node.right, node, depth + 1, x, y);
}
```

**Коли застосовувати:** задачі на пошук взаємозв'язків між вузлами в дереві (кузени, сіблінги, предки), де важлива комбінація глибини (рівня) та безпосереднього батька.

**Складність:** O(N) час, де N — кількість вузлів у дереві, оскільки у найгіршому випадку потрібно обійти все дерево. O(H) або O(W) пам'ять для DFS (стек рекурсії) або BFS (черга), де H — висота дерева, W — максимальна ширина дерева.
