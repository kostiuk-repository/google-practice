# Easy_07_MinimumDepthOfTree — Довідка

## Техніка: Binary Tree Leaf Depth (DFS/BFS)

**Суть:** обчислити мінімальну глибину дерева — відстань від кореня до найближчого листка. Важливий нюанс: якщо у вузла є лише один нащадок (лівий або правий), то мінімальна глибина визначається не по нульовому нащадку, а по наявному, оскільки шлях має закінчуватися на листку (вузлі без нащадків).

```java
public int minDepth(TreeNode root) {
    if (root == null) return 0;
    if (root.left == null) return minDepth(root.right) + 1;
    if (root.right == null) return minDepth(root.left) + 1;
    return Math.min(minDepth(root.left), minDepth(root.right)) + 1;
}
```

**Коли застосовувати:** пошук найкоротшого шляху від кореня до листка дерева, знаходження висоти збалансованого дерева або перевірка умов досяжності першого листка.

**Складність:** O(N) час у найгіршому випадку (якщо дерево вироджене в лінійний список). O(H) пам'ять на стек рекурсії для DFS. При використанні BFS час може бути меншим в середньому, оскільки пошук припиняється одразу при знаходженні першого листка.
