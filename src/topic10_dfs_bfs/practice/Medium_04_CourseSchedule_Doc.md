# Medium_04_CourseSchedule — Довідка

## Техніка: Topological Sort (Kahn's / DFS Color Detection)

**Суть:** визначити, чи можна пройти всі курси, якщо деякі мають пререквізити (це задача пошуку циклу в орієнтованому графі). Використовується алгоритм Кана (BFS по вхідних ступенях) або пошук циклів через DFS (фарбування вузлів у 3 кольори: білий — невідвіданий, сірий — в процесі обходу, чорний — повністю відвіданий).

```java
// Приклад DFS з кольорами для виявлення циклів
boolean hasCycle(int node, List<List<Integer>> adj, int[] visited) {
    if (visited[node] == 1) return true; // Сірий вузол - знайдено цикл!
    if (visited[node] == 2) return false; // Чорний вузол - оброблено
    visited[node] = 1; // Маркуємо сірим
    for (int next : adj.get(node)) {
        if (hasCycle(next, adj, visited)) return true;
    }
    visited[node] = 2; // Маркуємо чорним
    return false;
}
```

**Коли застосовувати:** виявлення взаємозалежностей, планування задач, перевірка циклічних посиланнях у збірниках пакетів або білд-системах.

**Складність:** O(V + E) час та пам'ять, де V — кількість курсів, E — кількість залежностей.
