# Easy_08_MatrixBlockSum — Довідка

## Техніка: Двовимірні префіксні суми (2D Prefix Sum / 2D Range Query)

**Суть:** Побудова 2D префіксної матриці `P`, де `P[i][j]` дорівнює сумі елементів підматриці від `(0,0)` до `(i-1, j-1)`. Для будь-якого прямокутника з кутами `(r1, c1)` та `(r2, c2)` сума дорівнює: `sum = P[r2+1][c2+1] - P[r1][c2+1] - P[r2+1][c1] + P[r1][c1]`.

```java
int[][] P = new int[R + 1][C + 1];
for (int r = 0; r < R; r++) {
    for (int c = 0; c < C; c++) {
        P[r + 1][c + 1] = mat[r][c] + P[r][c + 1] + P[r + 1][c] - P[r][c];
    }
}
```

**Коли застосовувати:**
- При багаторазових прямокутних запитах на статичній 2D матриці.

**Складність:**
- Часова складність: O(R * C) для побудови та обчислення результату.
- Просторова складність: O(R * C) для префіксної матриці.

## Вправи (Drills)
1. **01 — Compute 2D Prefix Sum**: Побудова 2D префіксної матриці.
2. **02 — Query 2D Region**: Запит суми регіону за готовою префіксною матрицею.
3. **03 — Row Prefix Sums**: Обчислення префіксних сум по рядках.
4. **04 — Column Prefix Sums**: Обчислення префіксних сум по стовпцях.
5. **05 — Complete Matrix Block Sum**: Розрахунок матриці блочних сум.
