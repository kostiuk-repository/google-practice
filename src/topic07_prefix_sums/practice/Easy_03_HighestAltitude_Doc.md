# Easy_03_HighestAltitude — Довідка

## Техніка: Префіксні суми (Prefix Sum)

**Суть:** Обчислення накопичених сум зліва направо для знаходження пікових значень.

```java
public int largestAltitude(int[] gain) {
    int maxAltitude = 0;
    int currentAltitude = 0;
    for (int g : gain) {
        currentAltitude += g;
        maxAltitude = Math.max(maxAltitude, currentAltitude);
    }
    return maxAltitude;
}
```

**Коли застосовувати:**
- Коли потрібно знайти максимальну або мінімальну накопичену суму під час покрокових змін.
- Коли дано відносні зміни значень, і необхідно відновити поточні значення.

**Складність:**
- Часова складність: O(N) — один прохід по масиву змін.
- Просторова складність: O(1) — використовується лише кілька змінних.

2. **02 — Lowest Altitude**: Знаходить найменшу досягнуту висоту (може бути від'ємною).
3. **03 — Altitude at Step K**: Знаходить висоту на конкретному кроці K.
4. **04 — Count Below Sea Level**: Рахує кількість кроків, на яких висота була нижче 0.
5. **05 — Max Gain In Single Step**: Знаходить максимальне одинарне збільшення висоти між кроками.
