# Easy_02_FindMiddleIndex — Довідка

## Техніка: Left/Right Sum Decomposition через префіксну суму

**Суть:** для кожного індексу `i` сума ліворуч (`leftSum`) — це префіксна
сума до `i` (не включаючи `i`), а сума праворуч (`rightSum`) — це
`totalSum - leftSum - nums[i]`. Обчислюємо `totalSum` один раз, потім
проходимо масив зліва направо, інкрементуючи `leftSum` і на кожному
кроці порівнюючи `leftSum` з `rightSum` без повторного підсумовування.

```java
int total = 0;
for (int x : nums) total += x;
int leftSum = 0;
for (int i = 0; i < nums.length; i++) {
    int rightSum = total - leftSum - nums[i];
    if (leftSum == rightSum) return i;
    leftSum += nums[i];
}
```

**Коли застосовувати:** будь-яка задача про "точку балансу" в масиві —
знайти індекс, де сума ліворуч дорівнює (або пов'язана певним
співвідношенням з) сумою праворуч. Ідентична за суттю задачі Find
Pivot Index — обидві розкладають масив на дві частини відносно
одного індексу через префіксну суму.

**Складність:** O(N) час (один прохід для totalSum, один для пошуку),
O(1) додаткової пам'яті.
