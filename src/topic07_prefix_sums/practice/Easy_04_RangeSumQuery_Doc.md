# Easy_04_RangeSumQuery — Довідка

## Техніка: Префіксні суми для інтервальних запитів (Range Sum Query)

**Суть:** Обчислення допоміжного масиву префіксних сум `P`, де `P[i]` зберігає суму перших `i` елементів. Це дозволяє відповісти на запит суми відрізка `[L, R]` за O(1) за формулою: `sum(L, R) = P[R + 1] - P[L]`.

```java
public class NumArray {
    private int[] prefix;

    public NumArray(int[] nums) {
        prefix = new int[nums.length + 1];
        for (int i = 0; i < nums.length; i++) {
            prefix[i + 1] = prefix[i] + nums[i];
        }
    }

    public int sumRange(int left, int right) {
        return prefix[right + 1] - prefix[left];
    }
}
```

**Коли застосовувати:**
- При багаторазових запитах сум відрізків масиву, який не змінюється (статичний масив).

**Складність:**
- Часова складність: O(N) для предобчислення, O(1) для кожного запиту.
- Просторова складність: O(N) для зберігання префіксних сум.

## Вправи (Drills)
1. **01 — Range Sum from Start**: Запит суми від початку масиву до індексу `right`.
2. **02 — Build Prefix Sum**: Побудова масиву префіксних сум за вихідним масивом.
3. **03 — Basic Range Sum**: Обчислення суми інтервалу за допомогою готового префіксного масиву.
4. **04 — Range Sum With Step**: Обчислення суми елементів на проміжку з кроком (наприклад, кожен 2-й елемент).
5. **05 — Range Sum Safe**: Запит суми з безпечною обробкою індексів поза межами масиву.
