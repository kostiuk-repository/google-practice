# Medium_05_CarPooling — Довідка

## Техніка: Масив різниць / Алгоритм сканування (Difference Array / Scanline Algorithm)

**Суть:** Оновлення інтервалів за O(1) за допомогою масиву різниць `diff`. Якщо пасажири заходять у пункті `from` та виходять у пункті `to`, ми додаємо пасажирів: `diff[from] += passengers` та віднімаємо: `diff[to] -= passengers`. Після цього ми робимо лінійне сканування (префіксну суму) для визначення кількості пасажирів на кожному відрізку.

```java
public boolean carPooling(int[][] trips, int capacity) {
    int[] diff = new int[1001];
    for (int[] trip : trips) {
        diff[trip[1]] += trip[0];
        diff[trip[2]] -= trip[0];
    }
    int current = 0;
    for (int change : diff) {
        current += change;
        if (current > capacity) return false;
    }
    return true;
}
```

**Коли застосовувати:**
- Коли є велика кількість інтервальних оновлень (додавання значень на проміжках) та потреба перевірити фінальний чи проміжний стан.

**Складність:**
- Часова складність: O(N + D), де N — кількість поїздок, D — максимальний пункт призначення.
- Просторова складність: O(D) для масиву різниць.
