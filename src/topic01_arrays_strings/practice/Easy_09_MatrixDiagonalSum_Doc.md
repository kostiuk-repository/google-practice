# Easy_09_MatrixDiagonalSum — Довідка

## Техніка: Diagonal Index Formula

**Суть:** для квадратної матриці розміру n x n елементи головної
діагоналі адресуються формулою `mat[i][i]`, а елементи побічної
діагоналі — формулою `mat[i][n-1-i]`, де `i` пробігає від `0` до
`n-1`. Це дозволяє звертатись до діагональних елементів напряму, без
вкладеного циклу по всій матриці. Коли `n` непарне, обидві діагоналі
перетинаються в центральному елементі (`i == n-1-i`), і його треба
рахувати лише один раз.

```java
int sum = 0;
int n = mat.length;
for (int i = 0; i < n; i++) {
    sum += mat[i][i];
    if (i != n - 1 - i) { // guard проти подвійного підрахунку центру
        sum += mat[i][n - 1 - i];
    }
}
```

**Коли застосовувати:** будь-яка задача про головну/побічну діагональ
квадратної (або прямокутної) матриці — сума, симетрія, транспонування,
модифікація діагональних елементів. Ключова ознака: індекси рядка й
стовпця пов'язані простою формулою (`i == j` або `j == n-1-i`), тож
немає потреби перевіряти всі `n*n` елементів.

**Складність:** O(N) час, де N — розмір діагоналі (для квадратної
матриці N = n, для прямокутної N = min(m, n)); O(1) додаткової пам'яті
поза виходом.

## Drills

1. **Easy_09_01** — сума лише головної діагоналі (mat[i][i]).
2. **Easy_09_02** — сума лише побічної діагоналі (mat[i][n-1-i]).
3. **Easy_09_03** — перевірка симетричності матриці відносно головної діагоналі.
4. **Easy_09_04** — обнулити головну діагональ на місці (мутація замість суми).
5. **Easy_09_05** — узагальнення формули на прямокутну (не обов'язково квадратну) матрицю.

Розв'язуй у порядку 01 → 05, не підглядаючи в Easy_09_MatrixDiagonalSum.java.
