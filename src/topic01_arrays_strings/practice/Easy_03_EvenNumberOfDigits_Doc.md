# Easy_03_EvenNumberOfDigits — Довідка

## Техніка: Підрахунок цифр числа (Digit Counting) + агрегація по масиву

**Суть:** щоб порахувати кількість цифр у числі, можна або перетворити
його в рядок (`String.valueOf(n).length()`), або ділити число націло
на 10 у циклі, доки воно не стане нулем, рахуючи кількість ітерацій.
Другий підхід не створює проміжних об'єктів-рядків і працює швидше.
Після того як є функція підрахунку цифр одного числа, вона
застосовується до КОЖНОГО елемента масиву в одному проході, а
результат агрегується — лічильником, новим масивом або булевим
прапорцем.

```java
int countDigits(int n) {
    if (n == 0) return 1;
    int count = 0;
    while (n > 0) {
        n /= 10;
        count++;
    }
    return count;
}
```

**Коли застосовувати:** будь-яке завдання, де потрібно щось знати про
"кількість розрядів" числа — парність/непарність кількості цифр,
порівняння чисел за довжиною запису, перевірка "чи вписується число в
N розрядів". Часто комбінується з простим проходом по масиву (map,
filter, count).

**Складність:** O(D) для підрахунку цифр одного числа (D — кількість
цифр), O(N * D) для всього масиву, O(1) додаткової пам'яті (окрім
випадків, коли результат — новий масив).

## Drills

1. **Easy_03_01** — знайти найбільше число в масиві і повернути кількість його цифр.
2. **Easy_03_02** — порахувати, скільки чисел мають НЕПАРНУ кількість цифр (дзеркало оригіналу).
3. **Easy_03_03** — повернути новий масив, де кожен елемент — кількість цифр відповідного числа (map).
4. **Easy_03_04** — перевірити, чи всі числа в масиві мають однакову кількість цифр.
5. **Easy_03_05** — порахувати, скільки чисел мають рівно 3 цифри.

Розв'язуй у порядку 01 → 05, не підглядаючи в Easy_03_EvenNumberOfDigits.java.
