# Medium_02_SpiralMatrix — Довідка

## Техніка: Чотири межі, що звужуються (Shrinking Boundary Pointers)

**Суть:** обхід матриці по спіралі підтримується чотирма вказівниками меж —
`top`, `bottom`, `left`, `right` — які окреслюють ще не відвіданий
прямокутник матриці. На кожному "колі" спіралі проходимо чотири сторони
цього прямокутника (наприклад, зліва направо по верхньому рядку, зверху
вниз по правому стовпцю, справа наліво по нижньому рядку, знизу вгору по
лівому стовпцю) і після кожної сторони зсуваємо відповідну межу всередину.
Перед третім і четвертим боком ОБОВ'ЯЗКОВО перевіряємо `top <= bottom` і
`left <= right`, інакше при виродженій (одно-рядковій/одно-стовпцевій)
матриці елементи будуть пройдені двічі або вказівник вийде за межі масиву.

```java
int top = 0, bottom = m - 1, left = 0, right = n - 1;
while (top <= bottom && left <= right) {
    for (int j = left; j <= right; j++) visit(top, j);
    top++;
    for (int i = top; i <= bottom; i++) visit(i, right);
    right--;
    if (top <= bottom) {
        for (int j = right; j >= left; j--) visit(bottom, j);
        bottom--;
    }
    if (left <= right) {
        for (int i = bottom; i >= top; i--) visit(i, left);
        left--;
    }
}
```

**Коли застосовувати:** будь-який обхід матриці "по колу, що звужується" —
читання чи запис елементів у спіральному порядку, обхід лише зовнішнього
кільця, обхід з іншої стартової точки чи в іншому напрямку, або обхід із
ранньою зупинкою після k елементів.

**Складність:** O(M·N) час (кожен елемент відвідується рівно один раз),
O(1) додаткової пам'яті (не рахуючи виходу).

## Drills

1. **Medium_02_01** — спіральний обхід проти годинникової стрілки (дзеркальна послідовність напрямків: вниз, вправо, вгору, вліво).
2. **Medium_02_02** — генерація n x n матриці, заповненої 1..n² за годинниковою спіраллю (та сама техніка, але в напрямку запису, а не читання).
3. **Medium_02_03** — лише зовнішнє кільце матриці за годинниковою стрілкою, без звуження меж всередину (один прохід чотирьох сторін).
4. **Medium_02_04** — спіраль, що починається з нижнього правого кута і йде проти годинникової стрілки до центру.
5. **Medium_02_05** — перші k елементів у спіральному (за годинниковою стрілкою) порядку з ранньою зупинкою.

Розв'язуй у порядку 01 → 05, не підглядаючи в Medium_02_SpiralMatrix.java.
