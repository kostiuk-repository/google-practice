 # Easy_02_PlusOne — Довідка

## Техніка: Carry / Borrow Propagation

**Суть:** масив цифр читається справа наліво (від молодшого розряду).
На кожному кроці рахуємо `digit + carry` (або `digit - borrow`),
записуємо `result % 10`, переносимо `result / 10` як carry в наступний
(лівіший) розряд. Якщо після проходу всіх розрядів carry ще лишився —
масив виріс на одну цифру.

```java
int carry = 1; // або k, або результат множення
for (int i = digits.length - 1; i >= 0 && carry > 0; i--) {
    int sum = digits[i] + carry;
    digits[i] = sum % 10;
    carry = sum / 10;
}
// якщо carry > 0 після циклу — потрібен новий масив на 1 довший
```

**Коли застосовувати:** "великі числа", які не влазять у `int`/`long`
і тому представлені масивом цифр — додавання, віднімання, множення на
малий множник, порівняння з рядковим/масивним представленням.

**Складність:** O(N) час, O(N) пам'ять у гіршому випадку (новий масив
+1 елемент), O(1) додаткової пам'яті поза виходом.
