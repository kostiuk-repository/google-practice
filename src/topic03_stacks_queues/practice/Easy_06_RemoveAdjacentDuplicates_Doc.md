# Easy_06_RemoveAdjacentDuplicates — Довідка

## Техніка: Стек символів для парного скасування (Pair Cancellation Stack)

**Суть:** Використання стеку для послідовного порівняння поточного символу з вершиною стеку. Якщо вони однакові, ми видаляємо символ зі стеку (скасовуємо пару), інакше — додаємо. Це дозволяє ефективно видаляти сусідні дублікати за один прохід.

```java
Deque<Character> stack = new ArrayDeque<>();
for (char c : s.toCharArray()) {
    if (!stack.isEmpty() && stack.peek() == c) {
        stack.pop();
    } else {
        stack.push(c);
    }
}
```

**Коли застосовувати:** Коли сусідні однакові елементи повинні взаємно знищуватися, і це знищення може викликати каскадний ефект для новостворених сусідів.

**Складність:** Часова: O(N), Просторова: O(N).
