# Medium_05_OddEvenLinkedList — Довідка

## Техніка: Розділення на два вказівники та зшивання (Multi-Pointer Partitioning)

**Суть:** підтримання кількох паралельних вказівників (наприклад, для непарних та парних елементів) для збирання окремих ланцюжків вузлів під час одного проходу, з подальшим зшиванням кінця першого ланцюжка з початком другого.

```java
if (head == null) return null;
ListNode odd = head;
ListNode even = head.next;
ListNode evenHead = even;

while (even != null && even.next != null) {
    odd.next = even.next;
    odd = odd.next;
    even.next = odd.next;
    even = even.next;
}
odd.next = evenHead;
return head;
```

**Коли застосовувати:** перевпорядкування вузлів за певною ознакою (парність індексу, парність значення, знак елемента), стабільне розбиття (stable partition) списку на кілька груп без створення нових вузлів.

**Складність:** O(N) час, O(1) додаткової пам'яті.

## Drills

1. **Medium_05_01** — розділення вихідного списку на два окремі списки (непарні індекси та парні індекси).
2. **Medium_05_02** — стабільне розділення вузлів за ознакою знаку значення (від'ємні попереду додатних та нулів).
3. **Medium_05_03** — перевпорядкування вузлів за парністю їхнього значення (парні значення попереду непарних).
4. **Medium_05_04** — групування непарних індексів, за якими йдуть парні індекси в зворотному порядку.
5. **Medium_05_05** — капстоун: розділення списку на три частини за індексом за модулем 3 (0, 1, 2) та їх зшивання.

Розв'язуй у порядку 01 → 05, не підглядаючи в Medium_05_OddEvenLinkedList.java.
