# Medium_01_RemoveNthFromEnd — Довідка

## Техніка: Два вказівники із затримкою (Fast/Slow Pointer with Delay)

**Суть:** використання двох вказівників, де перший (`fast`) рухається вперед на `n` кроків раніше за другий (`slow`). Коли `fast` досягає кінця списку, `slow` вказуватиме саме на елемент, розташований на відстані `n` кроків від кінця.

```java
ListNode dummy = new ListNode(0, head);
ListNode fast = dummy;
ListNode slow = dummy;

// Зсуваємо fast вперед на n + 1 кроків
for (int i = 0; i <= n; i++) {
    fast = fast.next;
}

// Рухаємо обидва вказівники одночасно
while (fast != null) {
    fast = fast.next;
    slow = slow.next;
}

// slow.next - це n-й вузол з кінця. Видаляємо його:
slow.next = slow.next.next;
return dummy.next;
```

**Коли застосовувати:** пошук або видалення K-го елемента з кінця списку за один прохід, розбиття списку у відношенні 1:K, знаходження елементів на певній фіксованій відстані від кінця без використання додаткової пам'яті.

**Складність:** O(N) час, O(1) додаткової пам'яті.
