# Medium_04_SwapNodesInPairs — Довідка

## Техніка: Локальна перебудова зв'язків (Pairwise Link Redirection)

**Суть:** зміна вказівників зв'язків сусідніх вузлів для обміну їх місцями без копіювання чи зміни внутрішніх значень вузлів. Для зручності використовується `dummy` вузол, щоб уникнути окремої логіки для зміни голови списку.

```java
ListNode dummy = new ListNode(0, head);
ListNode prev = dummy;

while (prev.next != null && prev.next.next != null) {
    ListNode first = prev.next;
    ListNode second = prev.next.next;
    
    // Перестановка зв'язків
    first.next = second.next;
    second.next = first;
    prev.next = second;
    
    // Зсув для наступної пари
    prev = first;
}
return dummy.next;
```

**Коли застосовувати:** перестановка сусідніх елементів парами, зміна груп елементів фіксованого розміру (наприклад, трійками), розгортання парних підмножин.

**Складність:** O(N) час, O(1) додаткової пам'яті.

## Drills

1. **Medium_04_01** — обмін місцями лише першої пари вузлів у списку.
2. **Medium_04_02** — обмін елементів парами через просту заміну значень (val) без зміни посилань (next).
3. **Medium_04_03** — рекурсивний підхід до обміну сусідніх вузлів у парах.
4. **Medium_04_04** — обмін сусідніх пар із пропуском кожного третього вузла (swap-swap-skip).
5. **Medium_04_05** — капстоун: обмін парами з додаванням вузла padding (-1) в кінець списку, якщо його довжина непарна.

Розв'язуй у порядку 01 → 05, не підглядаючи в Medium_04_SwapNodesInPairs.java.
