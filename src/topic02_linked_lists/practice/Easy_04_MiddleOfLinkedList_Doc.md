# Easy_04_MiddleOfLinkedList — Довідка

## Техніка: Slow/Fast Pointers (1x/2x Speed)

**Суть:** два вказівники стартують з `head`. `slow` рухається на 1
вузол за ітерацію, `fast` — на 2. Коли `fast` (або `fast.next`)
досягає `null`, `slow` перебуває рівно посередині списку. Для
парної довжини `slow` природньо зупиняється на ДРУГІЙ із двох середніх
вершин (через умову циклу `fast != null && fast.next != null`).

```java
ListNode slow = head, fast = head;
while (fast != null && fast.next != null) {
    slow = slow.next;
    fast = fast.next.next;
}
return slow; // середина
```

**Коли застосовувати:** будь-яка задача, де потрібна позиція,
пропорційна довжині списку, без попереднього підрахунку довжини за
окремий прохід — середина, поділ навпіл, k-та частка списку. Ключова
ознака: співвідношення швидкостей `fast`/`slow` визначає, ЯКУ частку
списку `slow` пройде до моменту, коли `fast` дістанеться кінця.

**Складність:** O(N) час, один прохід (замість двох: підрахунок довжини
+ друга навігація); O(1) додаткової пам'яті.
