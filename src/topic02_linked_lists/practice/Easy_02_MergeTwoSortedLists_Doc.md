# Easy_02_MergeTwoSortedLists — Довідка

## Техніка: Two-Pointer Merge + Dummy Head Splicing

**Суть:** заводимо `dummy`-вузол і хвостовий вказівник `tail`, що на
нього вказує. На кожному кроці порівнюємо голови двох списків і
приєднуємо (splice, а не копіюємо) вузол із меншим (або іншим за
критерієм) значенням до `tail.next`, після чого просуваємо `tail` і
відповідний вхідний вказівник. Коли один зі списків закінчився —
приєднуємо залишок іншого одним рухом.

```java
ListNode dummy = new ListNode(0);
ListNode tail = dummy;
while (a != null && b != null) {
    if (a.val <= b.val) { tail.next = a; a = a.next; }
    else                { tail.next = b; b = b.next; }
    tail = tail.next;
}
tail.next = (a != null) ? a : b;
return dummy.next;
```

**Коли застосовувати:** об'єднання двох (чи більше) впорядкованих
послідовностей в одну впорядковану, без виділення нових вузлів —
merge-крок сортування злиттям, дедуплікація під час злиття,
альтернативне "зшивання" двох списків.

**Складність:** O(N + M) час (по одному проходу кожного списку), O(1)
додаткової пам'яті — переиспользуємо існуючі вузли.
