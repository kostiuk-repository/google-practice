# Easy_03_LinkedListCycle — Довідка

## Техніка: Floyd's Tortoise and Hare (Fast/Slow Pointers)

**Суть:** два вказівники рухаються одним і тим же списком з різною
швидкістю: `slow` — на 1 крок, `fast` — на 2 кроки за ітерацію. Якщо в
списку є цикл, `fast` рано чи пізно "наздожене" `slow` всередині циклу
(відстань між ними скорочується на 1 щоразу). Якщо циклу немає, `fast`
дістанеться `null` першим.

```java
ListNode slow = head, fast = head;
while (fast != null && fast.next != null) {
    slow = slow.next;
    fast = fast.next.next;
    if (slow == fast) return true; // зустрілись -> є цикл
}
return false;
```

Друга фаза (знаходження ТОЧКИ входу в цикл): після зустрічі
поставити один вказівник назад у `head`, рухати обидва по 1 кроку —
вони зустрінуться рівно на вузлі, з якого починається цикл. Це
класичний математичний результат (відстань від голови до входу в цикл
дорівнює відстані від точки зустрічі до входу в цикл, якщо йти далі
циклом).

**Коли застосовувати:** виявлення циклів у зв'язних структурах (списки,
функціональні графи `next[i]`), пошук точки входу в цикл, довжина
циклу, а також будь-яка задача "чи два вказівники з різною швидкістю
колись зустрінуться" — розуміння того, ЧОМУ саме співвідношення 2x/1x
працює, критично важливе.

**Складність:** O(N) час, O(1) додаткової пам'яті — головна перевага
над підходом із `HashSet` відвіданих вузлів (O(N) пам'яті).
