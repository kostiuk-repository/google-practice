# Medium_03_AddTwoNumbers — Довідка

## Техніка: Порозрядне додавання з переносом (Digit-by-Digit Addition with Carry)

**Суть:** синхронна ітерація по двох числах, представлених списками розрядів (зазвичай у зворотному порядку), з обчисленням суми відповідних розрядів та переносом (carry) у наступний розряд.

```java
ListNode dummy = new ListNode(0);
ListNode curr = dummy;
int carry = 0;

while (l1 != null || l2 != null || carry != 0) {
    int val1 = (l1 != null) ? l1.val : 0;
    int val2 = (l2 != null) ? l2.val : 0;
    
    int sum = val1 + val2 + carry;
    carry = sum / 10;
    
    curr.next = new ListNode(sum % 10);
    curr = curr.next;
    
    if (l1 != null) l1 = l1.next;
    if (l2 != null) l2 = l2.next;
}
return dummy.next;
```

**Коли застосовувати:** додавання, віднімання або множення великих чисел довільної довжини, робота з різними основами числення, злиття числових послідовностей.

**Складність:** O(max(N, M)) час, O(max(N, M)) додаткової пам'яті для результуючого списку.

## Drills

1. **Medium_03_01** — додавання одноцифрового числа до великого числа у зворотному порядку.
2. **Medium_03_02** — додавання двох списків однакової довжини без переносу розрядів (спрощена модель).
3. **Medium_03_03** — множення числа, представленого у зворотному порядку, на 2.
4. **Medium_03_04** — додавання чисел у прямому порядку розрядів (від старшого до молодшого).
5. **Medium_03_05** — капстоун: додавання чисел з основою числення 100 (кожен вузол містить число від 0 до 99).

Розв'язуй у порядку 01 → 05, не підглядаючи в Medium_03_AddTwoNumbers.java.
